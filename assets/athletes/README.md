# Athlete Images

Place athlete photos here, named to match the `image` paths in `athletes.json`.

## Recommended Format
- Format: JPEG or WebP
- Size: 600×450 px (displayed at ~300×225)
- Max file size: ~80KB per image

## Naming Convention
Match the filename to the athlete's JSON entry, e.g.:
```
assets/athletes/marcus-thorne.jpg
assets/athletes/elena-vance.jpg
assets/athletes/jordan-rivers.jpg
```

## Image Sources
- High-res press photos from official team/agency channels
- Head-and-shoulders or half-body crop works best
- Neutral or action shot backgrounds

## Compression
Use [Squoosh.app](https://squoosh.app) to compress without noticeable quality loss.
